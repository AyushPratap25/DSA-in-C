# DSA in C

A comprehensive collection of Data Structures and Algorithms implemented from scratch in C. Built alongside semester coursework.

## Tech Stack

- **Language:** C
- **Editor:** VS Code

## Topics Covered

### Linked List
- Singly Linked List
- Doubly Linked List
- Circular Singly Linked List
- Circular Doubly Linked List
- SLL Functions & Applications (Polynomial Representation, Stack using Linked List)

### Stack
- Stack using Structure
- Stack using Linked List (Dynamic)
- Stack Sorting via Recursion
- Applications: Infix to Postfix Conversion, Postfix Expression Evaluation

### Queue
- Linear Queue
- Circular Queue

### Recursion
- Factorial, Power of Int
- Sum of Numbers
- Decimal to Binary
- String operations (Display, Reverse, Length)
- Linked List via Recursion (Length, Sum of Elements)

### Searching
- Linear Search (Sorted & Unsorted Array)
- Binary Search

### Sorting
- Bubble Sort (Array & Linked List)
- Insertion Sort
- Selection Sort
- Merge Sort (Arrays & Sub-arrays)

### Tree
- Tree Creation & Traversal (Inorder, Preorder, Postorder — Recursive & Non-Recursive)
- Binary Search Tree (Insertion, Search, Deletion)
- Expression Tree (via Postfix Expression)
- AVL Tree (Insertion — Iterative & Recursive, Deletion, AVL Check)

### Graph
- Adjacency Matrix Representation
- BFS Traversal (V1, V2)
- DFS Traversal (V1, V2)

## How to Run

Each topic has its own folder with `.c` files. Compile and run individually:

```bash
gcc filename.c -o output
./output
```
